# 🧠 Churn Prediction using ANN

## 📌 Project Overview
Predicting customer churn using an Artificial Neural Network (ANN) 
to help businesses retain customers proactively.

## 📊 Dataset
- Customer data with behavioral and demographic features
- Binary target: Churned (1) or Not Churned (0)

## 🛠️ Tools & Technologies
- Python
- Pandas & NumPy
- Scikit-learn
- TensorFlow / Keras
- Matplotlib & Seaborn

## ⚙️ Steps
1. Data Cleaning & Preprocessing
2. Encoding categorical variables
3. Handling Missing Values
4. Feature Scaling (Normalization)
5. Building ANN Model
6. Model Evaluation

## 📈 Model Evaluation
- Accuracy
- Confusion Matrix
- Precision & Recall

## 💼 Business Impact
Early identification of customers likely to leave,
enabling proactive retention strategies.
